# Interview Experiences – Sameer Wanjari

Hi! I’m **Sameer Wanjari**, a 5th-year student in the **Integrated Dual Degree (IDD) Engineering Physics** program.

I created this repository to share my placement interview experiences and help juniors prepare more effectively for the placement season. The aim is to provide a practical understanding of the entire recruitment process — from Online Assessments (OA) to technical and HR interview rounds.

Each company’s experience is documented separately in markdown files, making it easy to navigate and focus on the companies or roles you are interested in.
